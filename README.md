# **MoCap2GT: High-Precision Ground-Truth Estimator for SLAM Benchmarking**

**MoCap2GT** is a **motion-capture (MoCap)–based high-precision ground-truth (GT) trajectory estimator**, designed specifically for **SLAM benchmarking**. 
By directly fusing **MoCap data** with **IMU measurements** from the device under test (DUT), it enables:

- **Accurate spatiotemporal calibration** between the MoCap system and the DUT
- **Suppression of inherent high-frequency MoCap jitter**
- **Robust generation** of high-precision ground-truth trajectories
